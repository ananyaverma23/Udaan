# Udaan - Healthcare & Fertility Center

Udaan is a specialized medical platform built with React 19 and Vite, focusing on Fetal Medicine, Fertility services, and Multispeciality OPD.

## Quick Start
* **Install Dependencies:** `npm install`
* **Development Mode:** `npm run dev`
* **Build for Production:** `npm run build`

## Tech Stack
* **Framework:** React 19 & Vite 8
* **Routing:** React Router DOM 7
* **Styling:** Tailwind CSS 4 & PostCSS 8
* **Animations:** Framer Motion
* **Icons:** Lucide React & React Icons

## Key Features
* **Service Pages:** Comprehensive detail pages for IVF, IUI, Genetic Counselling, and Fetal Scans (NT, Anomaly, Growth, Doppler, etc.).
* **Patient Tools:** Integrated appointment booking system and "Scroll to Top" navigation.
* **Admin Dashboard:** Secure login to manage the medical gallery, including creating, editing, and deleting posts.
* **User Experience:** Features a global animated loader and responsive design for all devices.

## Project Structure
* **`src/pages/ServicePages`**: Specialized documentation for all medical treatments.
* **`src/pages/Admin`**: Management views for authentication and content updates.
* **`src/components`**: Reusable UI components including `Navbar`, `Footer`, `Hero`, and `Stats`.
* **`src/utils`**: Core utilities such as `galleryStorage.js` for data persistence.
